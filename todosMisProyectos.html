Este proyecto consiste en un juego inspirado en la tercera generación de Pokémon, desarrollado en Python. Incorpora características como la captura y entrenamiento de Pokémon, batallas por turnos y un sistema de niveles. Además, incluye un mapa básico y la posibilidad de interactuar con NPCs (personajes no jugables) para enriquecer la experiencia del usuario. Es una demostración de habilidades en la gestión de estructuras de datos complejas y en la implementación de lógica de juego, enfocada en crear experiencias de usuario envolventes.

Calculadora creada en Python capaz de realizar diversas operaciones matemáticas básicas, incluyendo suma, resta, multiplicación, división, así como cálculos más avanzados como potencias y raíces cuadradas. El diseño del proyecto incluye una interfaz de usuario simple y eficiente, con un código modular y bien documentado que facilita su mantenimiento y expansión. Este proyecto demuestra un manejo sólido de estructuras de control, funciones y gestión de excepciones en Python.

Dashboard interactivo que fue diseñado y desarrollado para el análisis detallado de campañas de e-mail marketing. Utilizando herramientas como Excel o Python, este dashboard visualiza métricas clave como tasas de apertura, clics, conversiones y segmentación de audiencia en tiempo real, facilitando la toma de decisiones basada en datos. El proyecto refleja una capacidad para transformar grandes volúmenes de datos en información procesable y presenta insights de manera clara y accesible, mejorando la efectividad de las estrategias de marketing.

Programa integral en Excel o Python que permite la gestión de la ingesta nutricional diaria, adaptándose a objetivos como pérdida de peso, aumento de masa muscular o mantenimiento. El software calcula la ingesta calórica según el perfil del usuario y sus metas, además de ayudar a planificar presupuestos de alimentos semanales y mensuales. Incluye recomendaciones personalizadas basadas en la ingesta de macronutrientes y un seguimiento de progreso, convirtiéndose en una herramienta esencial para quienes buscan mejorar su salud y bienestar.

Plataforma de e-commerce personalizada, diseñada para ofrecer una experiencia de compra fluida y eficiente. La web incluye un catálogo de productos dinámico, un sistema de carrito de compras y una pasarela de pagos segura que garantiza transacciones sin problemas. Además, se integraron funcionalidades como la gestión de inventarios, el seguimiento de pedidos y la personalización de la experiencia del usuario, facilitando la gestión eficiente de un negocio online y cumpliendo con las expectativas del cliente final.
